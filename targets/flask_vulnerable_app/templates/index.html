Vulnerabilidades Implementadas

Injeção SQL (SQLi)

Este aplicativo não sanitiza adequadamente as entradas em consultas SQL, permitindo ataques de injeção.

Cross-Site Scripting (XSS)

A aplicação não escapa corretamente o conteúdo gerado pelo usuário antes de renderizá-lo.

Execução Remota de Código (RCE)

Permite a execução de comandos do sistema operacional sem validação adequada.

Server-Side Request Forgery (SSRF)

A aplicação permite que um usuário controle URLs para as quais o servidor envia solicitações.

Desserialização Insegura

Utiliza o módulo 'pickle' do Python para desserializar dados não confiáveis.

Tokens Previsíveis

Algoritmos de geração de tokens com seed fixo, tornando-os previsíveis.
